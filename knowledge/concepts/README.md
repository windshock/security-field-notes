# Concepts

이 폴더는 보안 리서치 과정에서 반복적으로 등장하는 핵심 기술, 프로토콜, 공격 기법에 대한 **영구적인 지식 페이지**를 보관합니다.

## Purpose

- **지식 증류 (Distillation)**: 개별 리서치 아티클의 요약을 넘어, 해당 기술의 본질과 보안적 함의를 축적합니다.
- **중심점 (Pivot Point)**: 여러 리서치 노트, 탐지 가설, 실험 결과가 교차하는 허브 역할을 합니다.
- **성숙도 관리**: 기술에 대한 이해도가 높아짐에 따라 `Status`를 `evolving`에서 `stable`로 발전시킵니다.
