# Experiments

이 폴더는 보안 가설을 검증하기 위한 실험 계획과 결과를 기록합니다.

실험은 가능한 한 다음을 분리해서 작성합니다.

- 확인된 사실
- 전제
- 가설
- 실험 절차
- 관찰 결과
- 결론 강도
- 다음 실험

## Experiment Status

| Status | Meaning |
|---|---|
| planned | 아직 수행하지 않은 실험 |
| running | 진행 중 |
| observed | 결과 관찰 완료 |
| inconclusive | 결과가 불충분함 |
| validated | 가설이 충분히 검증됨 |
| rejected | 가설이 반박됨 |

## Naming Convention

```text
YYYY-MM-DD-topic-short-name.md
```

Example:

```text
2026-05-10-pla-dcom-local-collector-traces.md
```
